介绍bluevpn pro 客户端 windows 操作系统安装使用教程

首先，您应该拥有一个 bluevpn pro 账号，然后再简单配置一下客户端，即可畅享网络，正版 bluevpn pro 账号从 这里 获得。

从第二代起，bluevpn 统一使用 shadowsocks 作为客户端。

一、下载 Windows客户端

二、解压到本地目录（务必不要直接从压缩包运行），然后运行客户端

注意:如果XP下载后双击打不开，提示： 影梭 应用程序正常初始化失败。说明系统缺少.net framework

.NET Framework 2.0 版本，适合 Windows XP 用户，.NET Framework 3.5 版本，适合 Windows 7 用户，.NET Framework 4.0 版本，适合 Windows 8.1 用户，请自行通过搜索引擎搜索下载。

Windows8.1版本兼容微软最新发布的Windows10系统！

三、首次运行，弹出编辑服务器窗口，请输入您在 bluevpn官网 购买的服务，正确填写服务器地址、端口、密码和加密方式，或者直接扫描二维码，然后点确定。

在网站上找到您开通的服务器信息：

然后填写到对应的字段内：

如果您不知道如何填写，请关掉“编辑服务器”界面，然后点击页面上线路后面对应的二维码，此时不要关闭页面，然后在电脑任务栏找到小飞机图标，右键，从菜单中找到“扫描屏幕上的二维码”选项，直接扫码即可：

点确定后，右下角弹出提示

 

四、右键程序图标，弹出主菜单，勾选“启用系统代理”

五、打开任意浏览器（IE/Chrome/Firefox等），即可开始科学上网
